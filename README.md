# ALPHA-OS
Mac, mais pour vous.

L'accessibilité de Windows.
La beauté de Mac.

## Vision

ALPHA OS est une distribution Linux moderne basée sur Ubuntu/KDE
ayant pour objectif de proposer une interface cohérente, élégante
et performante.

## Objectifs

- Interface full dark
- Accent dynamique
- Dock moderne
- Multi-écran amélioré
- Animations fluides
- Paramètres unifiés
